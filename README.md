# Quant Finance Hub

An AI-powered learning platform for quantitative trading strategies.

Built with Next.js 16 (App Router), TypeScript, MongoDB, NextAuth.js, Tailwind CSS + shadcn-ui, and Groq AI.

**Live Demo:** (will be added after deployment)
**GitHub:** https://github.com/varsilsuvagiya/Quant-Finance-Hub

## Features

- User authentication (email/password)
- CRUD operations on Trading Strategies
- AI-powered strategy generation using Groq
- Responsive & accessible UI
- Type-safe code with Zod validation

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS + shadcn-ui
- MongoDB + Mongoose
- NextAuth.js (JWT)
- Tanstack Query (data fetching & caching)
- Groq SDK
- React Hook Form + Zod

## Setup

1. Clone the repo
2. `npm install`
3. Create `.env.local` with required variables
4. `npm run dev`
